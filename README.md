# tetsimu2_hub

[Tetsimu2](https://github.com/cuboktahedron/tetsimu2)の外部連携ツールです。


# 何ができるか

tetsimu2から外部のツールを呼び出したり、またその逆を行ったりします。  
現状は、[solution-finder](https://github.com/knewjade/solution-finder/)と連携して現在のフィールド状態からパフェ手順を探すことできます。
